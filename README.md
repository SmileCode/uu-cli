# uu-cli


一个快速构建项目的脚手架。  [github地址](https://github.com/xu455255849/vue-xu)


## 项目结构


```javascript
.
├── bin           //命令配置
├── README.md     //说明文档
├── lib           //功能文件
├── package.json  //包信息
└── test          //测试用例
```

## Install 
  
```javascript

  $ npm i uu-cli -g

```

## Usage

uu g [type] [name]

name为你需要生成的文件名
   
目前支持的type：
   
   * vue
   * angular
